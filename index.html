FoodShare

Web platform focusing on eliminating food waste through user to user sharing of food and leftovers. CRUD application using Node.js, Express, and MongoDB. User will be able upon login to post, edit, and delete food items and contact others in the community to eliminate food waste. App uses USDA API to return ingredients of items upon request by user. Project still in development but ongoing code can be accessed on github.
